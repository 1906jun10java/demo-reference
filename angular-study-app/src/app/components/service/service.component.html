A Service is a class that is not dependent on any particular component which is used to provide shared data, logic across components, and encapsulate external interactions.
Components leverage Services to enhance views.
We can have Services communicating with endpoints through HTTP.
They can act as the gateway between our middle and front end.
We inject services into components with Dependency Injection.
How to create a service:
  1. Create a class, decorate it with the @Injectable decorator, and export it.
  2. Register the provider. A provider is code that can create or return a service. Add the service to the providers property in either @Component (=> injectable to component and its children) or @NgModule (=> injectable everywhere within module).
  3. Inject the service into the desired component via its constructor (similar to Java, every class has an implicit noargs constructor and we may define a constructor with arguments to be used in its place). e.g. constructor(private myService: MyService)
